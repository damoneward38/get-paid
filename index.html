Platform Overview

Gifted Eternity is a comprehensive music streaming and distribution platform designed for artists, labels, and music enthusiasts. It combines music discovery, streaming, artist collaboration, and multi-platform distribution capabilities.

Built with modern web technologies, this platform showcases professional-grade backend services, database schema, and deployment configurations.

Project Structure

Gifted Eternity is a full-stack streaming and analytics platform.

Architecture

This is a full-stack application with three main components:

Frontend

React 19 + Vite + TailwindCSS for a modern, responsive user interface

Backend

Node.js + Express + tRPC for type-safe API procedures

Database

MySQL/TiDB with Drizzle ORM for data persistence

Key Features

  • Multi-Platform Distribution: Distribute to 8 DSPs (Spotify, Apple Music, Amazon, YouTube, TIDAL, Deezer, Pandora, SoundCloud)
  • Royalty Tracking: Real-time earnings and royalty management
  • Release Management: Schedule and manage music releases
  • Metadata Management: Complete track metadata and artwork management
  • Rights Management: Track ownership and rights across platforms
  • Distribution Analytics: Performance metrics and insights
  • Artist Collaboration: Team-based project management
  • Playlist Pitching: Submit to curated playlists
  • White Label Distribution: Custom branding options
  • PayPal Integration: Automatic artist payouts
  • Subscription management

Getting Started

Local Development


# Clone the repository
git clone https://github.com/damoneward38/G-E-Stream.git
cd G-E-Stream

# Install dependencies
pnpm install

# Set up environment variables
cp .env.example .env.local

# Run development server
pnpm dev
